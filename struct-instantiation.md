Struct instantiation
====================

-   struct-instantiation-expression:

    -   type-name **(** arguments **)** [ **{** initializer **}** ]

    initializer:

    -   field-name **=** expression [ **,** initializer ]

    arguments:

    -   expression [ **,** arguments ]
